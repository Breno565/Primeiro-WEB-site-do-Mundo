Veja os direitos autorais do CERN. Este é o arquivo LEIA-ME que você obtém quando desembrulha um dos nossos arquivos tar. Esses arquivos contêm informações sobre hipertexto, sistemas de hipertexto e o projeto WorldWideWeb. Se você tirou isso com um arquivo .tar, você terá apenas um subconjunto dos arquivos.

ESTE ARQUIVO É UMA VERSÃO MUITO ABREVIADA DAS INFORMAÇÕES DISPONÍVEIS NA WEB. EM CASO DE DÚVIDA, LEIA A WEB DIRETAMENTE. Se você não possui um navegador instalado, faça isso pelo telnet para info.cern.ch (sem nome de usuário ou senha).

Estrutura do diretório Archive

Em / pub / www, além deste arquivo LEIA-ME, você encontrará os diretórios bin, src e doc. Os arquivos principais são os seguintes:

  • src / WWWLibrary_v.vv.tar.Z

      • A biblioteca W3. Todas as fontes e Makefiles para sistemas selecionados.

  • src / WWWLineMode_v.vv.tar.Z

O navegador do modo Linha - todas as fontes e Makefiles para sistemas selecionados. Requer a biblioteca.

  • src / WWW / LineModeDefaults_v.vv.tar.Z

A doucmnetation on-line para o navegador de modo de linha, que você ainda pode desejar se obtiver o programa como binário. Um subconjunto da fonte completa acima.

  • src / WWWNextStepEditor.tar.Z

O Navegador / editor de hipertexto para o NeXT - fonte e binário.

  • src / WWWDaemon_v.vv.tar.Z

O daemon HTTP e os programas de gateway WWW-WAIS. Fonte.

  • doc / WWWBook.tar.Z

Um instantâneo de nossa documentação interna - preferimos que você acesse isso on-line - veja os avisos abaixo.

  • bin / xxx / bbbb

Binários executáveis ​​do programa bbbb para o sistema xxx. Verifique o que há antes de se preocupar em compilar. (Nota: HP é chamado de "cobra")

Servidores

O arquivo tar do Daemon contém (nesta versão) o código para o daemon HTTP básico para servir arquivos e também para o gateway WWW-WAIS. Para compilar o gateway WAIS, você precisará de [um link para] uma distribuição WAIS no mesmo nível que o diretório WWW.

Geral
