Isso descreve a política do projeto W3 no CERN. Embora não seja juridicamente vinculativo, isso tenta explicar meu entendimento das regras do CERN e os desejos da equipe do CERN.

Alvo

O objetivo básico do projeto é promover a disponibilidade de comunicação e informação para a comunidade de Física de Alta Energia (HEP). O projeto é baseado no CERN, cujo orçamento é fornecido por contribuições do dinheiro do contribuinte dos estados membros europeus. É do interesse do HEP, do CERN e do próprio projeto que ele interaja com sistemas e informações em muitos outros campos, para que a colaboração ativa com outros grupos seja essencial. Produzir um sistema de informação que isola o HEP do resto do mundo seria contraproducente, de modo que o objetivo pode ser visto como um avanço na rede global de informações. A equipe da WWW está entusiasmada com o fato de que informações de todos os tipos devem estar disponíveis o mais amplamente possível.

Colaboração

Incentivamos a colaboração de partes acadêmicas ou comerciais. Sempre há muitas coisas a serem feitas, portas a serem criadas para diferentes ambientes, novos navegadores a serem gravados e dados adicionais a serem incorporados na "web". Já houve muitas contribuições nesses termos e também com o suporte de hardware dos fabricantes. Se você estiver interessado em ampliar a Web ou o software, envie um email ou telefone.

Distribuição de código

O código escrito no CERN é coberto pelos direitos autorais do CERN. Na prática, a interpretação disso no caso do projeto W3 é que os programas estão disponíveis gratuitamente para os órgãos acadêmicos. Para organizações comerciais que não a revendem, mas a estão usando para participar do intercâmbio global de informações, a cobrança geralmente é dispensada para reduzir custos administrativos. É claro que o código é compartilhado livremente com todos os colaboradores. As organizações comerciais que desejam vender software com base no código W3 devem entrar em contato com o CERN. Estamos no processo de obter um acordo para liberar certas partes do código do projeto WWW com a Licença Pública Geral (e a Licença da Biblioteca GP).

Onde o código CERN estiver incluído em outro código de domínio público, esse código CERN também se tornará domínio público.

O código não originário do CERN é obviamente coberto pelos termos definidos pelo detentor dos direitos autorais envolvido. Protocolos e formatos de dados

A definição de protocolos, como HTTP, e formatos de dados, como HTML, são de domínio público e podem ser usados ​​livremente por qualquer pessoa
