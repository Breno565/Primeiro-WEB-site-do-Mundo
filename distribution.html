Leia isto se você ainda não obteve os arquivos WWW necessários e não pode montar o NFS em um diretório no qual eles residem. A origem e o binário dos arquivos do projeto WWW estão atualmente disponíveis (consulte direitos autorais) por FTP anônimo no nó info.cern.ch. Atualmente, este nó é 128.141.201.74, mas PODE MUDAR. Os arquivos são arquivos compactados em fita (.tar.Z). Para obter um arquivo usando FTP anônimo, use o nome de usuário "anonymous" e seu endereço de email como senha: -

  • > ftp info.cern.ch
  • Digite o nome do usuário: anônimo
  • Senha: me@myhost.edu
  • ftp> cd / pub / www / src
  • ftp> binário
  • ftp> obtenha WWWLineMode_vvv.tar.Z
  • ftp> sair>

O nome do arquivo dependerá do produto que você deseja. Aqui vvv é o número da versão do software. Depois de ter o arquivo tar compactado, você deve descompactá-lo e desembrulhá-lo.

descompacte WWWLineMode_vvv.tar.Z

Isso criará um arquivo bastante maior com o nome sem o ".Z". Para desembrulhá-lo, tar xvf WWWLineMode_vvv.tar

Agora você deve instalar o navegador de modo de linha, instalar o servidor, instalar o Viola, etc.
