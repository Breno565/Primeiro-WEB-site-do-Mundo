Biblioteca comum

A biblioteca de códigos comum é a base para a maioria dos navegadores. Ele contém o código de acesso à rede e a manipulação de formatos.

Clientes

Navegador de modo de linha

Este programa fornece leitores W3 a qualquer pessoa com um terminal burro. Não é tão chamativo quanto uma implementação de janelas, abrange uma ampla classe de usuários que ainda não possuem recursos de janelas e é uma ferramenta de recuperação de informações de uso geral.

Amaldiçoa o Navegador

Assume um terminal de grade de caracteres com endereçamento do cursor e fornece uma interface de tela cheia para a web.

Servidores

Servidor VM

Um servidor para mainframes da VM.

Servidor genérico

Um servidor para sistemas unix e VMS. Um servidor de arquivos rápido, adequado à documentação on-line e adaptável a muitas necessidades.

Gateway VMS / Ajuda

Execute um computador VMS e permita que o cliente da web leia todos os arquivos locais do VMS / Ajuda.

Robô de correio

Um servidor que retornará documentos por correio, mediante uma solicitação enviada por correio. Também gerencia listas de discussão.
