Qual programa você está usando para o qual deseja ajuda online?

Navegador de modo de linha

Usando o navegador de modo de linha, selecione uma opção com um número e pressione RETURN

Viola

Para obter ajuda sobre o Viola, basta pressionar o ícone de ponto de interrogação (?) Próximo ao canto superior direito.

Próxima Etapa

Para selecionar ajuda com este navegador, use a Ajuda no menu Informações ou selecione esta opção com o mouse (clique duas vezes).

MidasWWW

Para obter ajuda sobre este navegador, selecione esta opção com o mouse.

Consulte também o manual do usuário do servidor W3. Para outras perguntas, navegue na página do projeto W3.
