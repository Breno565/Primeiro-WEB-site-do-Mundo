Geral

Quando você clica duas vezes no ícone WorldWideWeb, o aplicativo por padrão exibe sua própria página inicial, ~ / WWW / default.html, ou, na sua falta, na página inicial do sistema, /usr/local/lib/WWW/default.html. A partir de então, se você quiser selecionar um documento referido por uma página que está sendo exibida, basta clicar duas vezes em uma área de texto em cinza. (O cinza foi escolhido para não usar outras técnicas de destaque, como negrito ou itálico, que os autores podem querer usar para outras coisas. Quando a cor aparecer, podemos usar a cor ...

Se você clicar duas vezes em qualquer outro lugar do documento, selecionará uma palavra, como é habitual no NeXT.

Algumas áreas cinzas levam a documentos inteiros, enquanto outras exibem um documento com uma área de destino específica realçada. Isso é chamado de âncora de destino. Não fica cinza, a menos que por sua vez leve a outro lugar, mas clicar duas vezes em uma âncora de destino seleciona a âncora inteira.

Índices

Certos documentos são índices. Eles normalmente têm apenas uma pequena quantidade de texto. Você não pode ler o índice por conta própria (pode ser muito grande). Para acessar algo através do índice, use a opção de menu "Pesquisa por palavra-chave". A janela com o índice deve ser a janela principal (barra de título escura) quando você usa o painel de pesquisa de palavras-chave. As regras usadas pelos índices para procurar correspondências variam. Índices diferentes usam técnicas diferentes e abrangem diferentes faixas de informações. Quando uma pesquisa é bem-sucedida, o resultado é uma página de referências a documentos. Você seleciona documentos clicando duas vezes da maneira usual.
